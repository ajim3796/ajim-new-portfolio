# 私について (About Me)

## プロフィール・経歴

Python (Django) を用いた EC 向け検索エンジン開発の経験を持つ、Web エンジニア。
バックエンドの設計から実装、そしてフロントエンド領域を独学で磨き上げた、フルスタックな視点が強みです。

### 強みと得意領域

- **バックエンド・システム設計:** 大規模 EC サイトの検索エンジン開発において、**API の開発**、**商品データの効率的な取り込み**、EC サイトとの**接続要件定義**など、**上流工程から開発・保守まで**幅広く対応してきました。システム全体を俯瞰した設計と課題解決が得意です。
- **フロントエンド・UI/UX へのこだわり:** 独学でモダンなフロントエンド技術を習得し、**視認性やパフォーマンスを最優先**した開発を実践しています。Chrome 拡張機能での視認性向上や、自作サイトでの**Base64 埋め込みによる画像の遅延ゼロ化**など、技術的な工夫を凝らした実装を重視しています。
